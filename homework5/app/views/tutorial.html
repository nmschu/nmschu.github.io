How this was made

This is a step by step on how I went about creating the "Home" section of this site. Going from the top down, this is how you are able to do it as well, even if you don't have any angular knowledge.



Starting the site

To start the site, you need to get a few things installed on your machine. I have a Mac, so all information given will be using a Mac for the example. But before we get too much into it, we need to download a few things.

NPM Install

NPM comes bundled in with node. So here is a link to download it. Choose the mature and dependable version. At time of writing this, that is version 4.3.0 LTS.

NPM is a package manager. Once this is installed, you can open terminal and type in a few words, and it will automatically download the other packages we need.

Bower Install

On the command line, type in: npm install -g bower

Grunt Install

On the command line, type in: npm install -g grunt-cli

Yeoman Install

On the command line, type in: npm install -g yo

Getting the generators

Generators are [EXPLAIN GENERATORS HERE!!!!!!!]

On the command line, type in: npm imstall --global generator-angular@0.11.1 generator-karma


Scaffolding

Now that we have all that, we are able to use Yeoman to scaffold a website for us. This way, we dont have to do too much of the set up and connection of pages. After its set up we can customize it to our heart's desire, but for now, we will leave it as it is.

Set up

Before we let Yeoman build it for us, we need to make a folder to put everything in. Create a folder on your desktop and name it whatever you want. Then, on the command line, type in: cd ["Folder Name"]. Now we are in the correct directory, and when we run Yeoman, all the files will be saved there.

Creating the site

Now that we are in the correct directory (the folder you just made), lets let Yeoman build it for us. On the command line, type in yo. It will ask you to select a generator. We are going to be building it in angular, so hit enter. Next, it is going to ask you some questions on how you want it set up, we will NOT be using Sass (with Compass), we WILL be using Grunt, and also using Bootstrap.

After that, it asks which modules you would like to include. Deselect them all (use the arrow keys to navigate, and the space bar to select/deselect), then hit enter. It will take a few minutes, then you will have a website built for you. You hardly had to do anything!
